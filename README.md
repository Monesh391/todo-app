# Todo Application

## Description
A full-stack Todo application built with React, Node.js, and Express. Users can create, view, update, and delete todos.

## Features
- View all todos
- View todo details
- Create a new todo
- Update a todo
- Delete a todo
- Mark todo as completed or pending

## Technologies Used

### Frontend
- React
- React Router
- Axios
- Vite

### Backend
- Node.js
- Express.js

## API Endpoints

GET /todos

GET /todos/:id

POST /todos

PUT /todos/:id

DELETE /todos/:id

## How to Run

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open:

```
http://localhost:5173
```
